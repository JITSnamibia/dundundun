Google Search Console Setup

To get indexed, your site must be verified in Google Search Console. This is non-negotiable.

1

Add your domain to Search Console

Use the 'Domain' property option for full coverage.

2

Get the verification TXT record

Google will give you a TXT record to add to your DNS settings on GoDaddy.

3

Wait and verify

After adding the record, wait for propagation, then click "Verify" in Search Console.

Creating a `robots.txt` File

A robots.txt file tells search engines which pages the crawler can or can't request. It's crucial for managing crawl budget and preventing unimportant pages from being indexed.

Basic `robots.txt` for a Canva Site

For most simple websites, you want Google to be able to crawl everything. A basic file that allows all access looks like this:

User-agent: *
Allow: /

Create a plain text file named `robots.txt` with this content and upload it to the root directory of your website (e.g., `yourdomain.com/robots.txt`).

Fixing 404s with Redirects

If you've deleted a page or changed its URL, the correct solution is to set up a 301 redirect. This forwards users and search engines from the old URL to the new one, preserving link equity.

Using `.htaccess` for Redirects

If your hosting provider uses an Apache server, you can add rules to your .htaccess file. To redirect an old page to a new one, add this line:

Redirect 301 /old-page.html https://www.yourdomain.com/new-page.html

Editing your `.htaccess` file can break your site if done incorrectly. Always make a backup before making changes.
